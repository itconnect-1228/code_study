# T083: TaskDetail 페이지

## 무엇을 만들었나요?

태스크 상세 정보를 보여주는 페이지를 만들었어요. 세 개의 탭(문서, 실습, Q&A)으로 구성되어 있고, 각 탭은 후속 Phase에서 구현될 기능의 뼈대 역할을 해요.

## 왜 이렇게 만들었나요?

태스크 상세 페이지는 AI 학습 플랫폼의 핵심이에요:
- **문서 탭**: AI가 생성한 학습 문서 (Phase 6)
- **실습 탭**: 연습 문제 풀이 (Phase 7)
- **Q&A 탭**: AI에게 질문하기 (Phase 8)

지금은 뼈대만 만들어두고, 나중에 각 기능을 채워 넣을 거예요.

## 어떻게 작동하나요?

1. 태스크 ID로 데이터를 불러와요
2. 헤더에 태스크 번호, 제목, 상태, 언어를 표시해요
3. 상태에 따라 다른 콘텐츠를 보여줘요:
   - **대기중**: "문서 생성을 기다리고 있습니다"
   - **생성중**: 로딩 스피너 + "AI가 코드를 분석하고 있습니다"
   - **완료**: "학습 문서가 준비되었습니다"
   - **오류**: 에러 메시지
4. 탭을 클릭해서 문서/실습/Q&A 사이를 이동할 수 있어요

## 배운 개념

- **Tabs 컴포넌트**: 탭 기반 UI 구현
- **상태 기반 렌더링**: 같은 탭에서 상태에 따라 다른 콘텐츠 표시
- **스켈레톤 로딩**: 데이터 로딩 중 보여주는 플레이스홀더

## 주의사항

- 현재는 플레이스홀더 콘텐츠만 있어요. 실제 기능은 후속 Phase에서 구현됩니다
- 상태가 'generating'일 때 아이콘에 회전 애니메이션이 적용돼요
- 존재하지 않는 태스크 접근 시 에러 메시지와 대시보드 링크를 표시해요

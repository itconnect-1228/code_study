# T079: TaskCard 컴포넌트

## 무엇을 만들었나요?

태스크 정보를 카드 형태로 보여주는 React 컴포넌트를 만들었어요. 프로젝트 상세 페이지에서 타임라인처럼 태스크 목록을 보여줄 때 사용돼요.

## 왜 이렇게 만들었나요?

사용자가 여러 개의 태스크를 한눈에 볼 수 있게, 핵심 정보를 카드 형태로 정리했어요:
- 태스크 번호 (Task 1, Task 2, ...)
- 제목
- 상태 (대기중, 생성중, 완료, 오류)
- 프로그래밍 언어
- 업로드 방식 (파일, 폴더, 붙여넣기)
- 생성 날짜

## 어떻게 작동하나요?

1. 태스크 데이터를 받아서 카드 형태로 렌더링해요
2. 상태에 따라 다른 색상의 배지를 보여줘요 (완료=초록, 오류=빨강 등)
3. 클릭하면 태스크 상세 페이지로 이동해요 (링크) 또는 콜백 함수 호출
4. 타임라인에서 마지막 항목이 아니면 연결선을 표시해요

## TDD로 어떻게 테스트했나요?

- 19개의 테스트 케이스 작성
- 렌더링, 상태별 표시, 업로드 타입별 표시, 클릭 동작, 스타일링 등 테스트
